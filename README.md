# DeliveryDrink

##Instalation

Clone this repo.

For the back directory in `src`, create a file `.env` directory next to `server.js` with this content:

    PORT=8080
    
    DB_HOST=localhost
    DB_USER=root
    DB_PASSWORD=password
    DB_NAME=db_name
    
And in directory `front` and `back`, run:

    npm install
    
Finish with run in root :

    npm install

    Read API.md in root
